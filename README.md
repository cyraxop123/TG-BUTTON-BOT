- `/start` --> to check bot ALIVE or NOT
- `/add` <YOUR TEXT> --> To Add Poll
- `/do` --> Starting bot poll
- `/clean` --> To clear polls options
- `/see` --> To see avalable poll option
- `/stopbot` --> to stop the poll
