# AVAILABLE COMMANDS
1. '/start' --> to check bot ALIVE or NOT
2. '/add' <YOUR TEXT> --> To Add Poll
3. '/do' --> Starting bot poll
4. '/clean' --> To clear polls options
5. '/see' --> To see avalable poll option
6. '/stopbot' --> to stop the poll
